📖 How to obtain a Vertex AI Service Account Key
  1. Open Google Cloud Console → Credentials
  2. Select your project (e.g. ai-project-384207)
  3. Click Create Credentials → Service Account
  4. Name your service account (e.g. vertex-monitor), click Create
  5. Assign role: Vertex AI User (or Vertex AI Administrator)
  6. After creation, click the account → Keys tab
  7. Click Add Key → Create New Key, choose JSON format
  8. Download the file, open it in a text editor, copy all content and paste above

⚠️ Security notice: The key is stored only inside the local Docker container and is never uploaded to external services.
